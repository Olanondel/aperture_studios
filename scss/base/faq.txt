1. fonts - подключение семейств шрифтов
2. typography - перечень всех вариантов типографии на проекте.
3. base.scss - базовые стили для html тегов 
   (не допускается стилизация глобальных тегов с классами)
4. breakpoint.scss - bootstrap breakpoint механизм  для типичного адаптива
5. mixins.scss - глобальные миксины для проекта
6. normalize.scss - присвоение общих стилей для всех тегов вне зависимости от браузера или платформы.
7. transitions.scss - базовые классы отвечающие за плавность (fade для модальных окон и тд).
8. variables.scss - глобальные переменные для проекта
